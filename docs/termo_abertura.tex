\section{Identificação do Projeto}
\begin{itemize}
    \item \textbf{Nome:} Repositório Digital Institucional UEMA
    \item \textbf{Versão:} 1.0
    \item \textbf{Data:} Dezembro 2025
    \item \textbf{Responsável:} Equipe de Desenvolvimento
\end{itemize}

\section{Justificativa}
A UEMA necessita de uma solução integrada para gestão, armazenamento e consulta de documentos e processos institucionais. Atualmente, os setores utilizam sistemas isolados, causando:
\begin{itemize}
    \item Dificuldade na localização de documentos
    \item Duplicidade de informações
    \item Falta de padronização
    \item Processos lentos e burocráticos
\end{itemize}

\section{Objetivo}
Desenvolver um sistema web para centralizar documentos e processos dos setores PROGEP, PROPLAD, PROEXAE, PROG, PPG e Protocolo Geral, com interface moderna, busca inteligente e assistente de IA.

\section{Escopo}
\subsection{Incluído}
\begin{itemize}
    \item Módulo de autenticação com níveis de acesso
    \item Gestão de documentos (upload, categorização, busca)
    \item Gestão de processos administrativos
    \item Dashboard com indicadores
    \item Assistente de IA para consultas
    \item Relatórios e exportação
\end{itemize}

\subsection{Excluído}
\begin{itemize}
    \item Integração com sistemas legados
    \item Aplicativo mobile nativo
    \item Assinatura digital avançada
\end{itemize}

\section{Stakeholders}
\begin{itemize}
    \item \textbf{Patrocinador:} Reitoria UEMA
    \item \textbf{Usuários finais:} Servidores dos setores
    \item \textbf{Equipe técnica:} Desenvolvedores do projeto
\end{itemize}

\section{Benefícios Esperados}
\begin{enumerate}
    \item Redução de 60\% no tempo de busca de documentos
    \item Centralização de informações institucionais
    \item Padronização de processos
    \item Melhoria na tomada de decisões
    \item Segurança e rastreabilidade
\end{enumerate}
